## 红线

出现下面情况时，不继续硬推原计划，先调整：

| 红线 | 处理 |
| --- | --- |
| 连续 3 天目标达成低于 3 分 | 下调任务量，保留核心任务 |
| 某科连续 2 周正确率不升 | 停止盲刷，回到知识点复盘 |
| 手机时间连续 3 天过高 | 晚上复盘时写触发原因 |
| 学习时长高但复盘空白 | 说明只是耗时间，要改学习方式 |
