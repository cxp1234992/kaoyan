# GitHub 网页使用方法

你以后可以不在本地改，直接在 GitHub 网页改这些文件。

## 怎么改

1. 打开仓库。
2. 进入 `我的考研系统` 文件夹。
3. 点开要改的文件。
4. 点右上角铅笔图标。
5. 改完后点 `Commit changes`。

## 每天推荐改哪几个

每天改：

- `01-每日记录.md`
- `02-学习时长.md`

做题后再改：

- `03-正确率.md`
- `04-错题复盘.md`

每周改：

- `05-周复盘.md`

每月改：

- `06-月复盘.md`

## 不要怕写少

刚开始每个文件只写一行都可以。真正重要的是持续留下痕迹。
