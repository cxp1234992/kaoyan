# 我的考研学习系统

这套模板保留原仓库最有用的逻辑：计划、时长、正确率、状态、复盘。

我把它改成更适合你自己从零开始用的版本。不要先追求复杂，先每天写三件事：

1. 今天学了什么。
2. 学了多久。
3. 哪些地方错了、卡了、下次怎么改。

## 每天怎么用

每天晚上打开这三个文件，补一行就够：

- `01-每日记录.md`
- `02-学习时长.md`
- `03-正确率.md`

如果当天没有做成套题，`03-正确率.md` 可以不写。

## 每周怎么用

每周日打开：

- `05-周复盘.md`

回答三个问题：

1. 哪科进步了？
2. 哪科只是花了时间但没效果？
3. 下周要砍掉什么、加强什么？

## 每月怎么用

每月底打开：

- `06-月复盘.md`

重点看趋势，不要只看某一天状态。

## 文件说明

| 文件 | 用途 |
| --- | --- |
| `00-总目标.md` | 目标院校、分数、阶段安排 |
| `01-每日记录.md` | 每天的计划、完成、状态 |
| `02-学习时长.md` | 记录各科真实学习时间 |
| `03-正确率.md` | 记录刷题、真题、阅读正确率 |
| `04-错题复盘.md` | 记录真正值得回看的错题 |
| `05-周复盘.md` | 每周调整节奏 |
| `06-月复盘.md` | 每月看大方向 |
| `07-资料清单.md` | 管理书、课、题、软件 |

## 这套系统的原则

- 记录要轻，不要把记录本身变成负担。
- 时间只能说明投入，正确率和复盘才说明质量。
- 每周必须调整计划，不要机械执行已经不适合自己的安排。
- 错题不是越多越好，只收录能暴露薄弱点的题。
- 状态差不等于今天废了，但状态连续差三天就必须改节奏。
